# Финальный QA Checklist

Перед демонстрацией портфолио проверьте:

- [ ] Quick Start выполняется без правок команд.
- [ ] Interactive API открывается и загружает `openapi.yaml` без `404`.
- [ ] Все внутренние ссылки работают.
- [ ] Диаграммы (BPMN, C4, UML) отображаются на страницах.
- [ ] Мобильная версия читаема (меню, код-блоки, таблицы).
- [ ] Темная и светлая тема переключаются корректно.
- [ ] Последние изменения отражены в `Changelog`.
- [ ] CI workflow зеленый на `main`.
- [ ] Версия `latest` доступна в переключателе версий.
