# Кейс: инструкция по скриншотам на macOS

Кейс показывает структуру пользовательской инструкции: от базового сценария к частным вариантам и ограничениям.

## Задача

Подготовить руководство для пользователя: как сделать скриншот на компьютере с macOS.

В инструкции важно было показать несколько сценариев:

- снимок всего экрана;
- снимок окна;
- снимок части экрана;
- снимок меню;
- работа через встроенное приложение Screenshot.

## Принцип структуры

Пользователь приходит не за описанием инструмента, а за результатом. Поэтому инструкция строится по задачам:

| Сценарий | Команда |
|---|---|
| Снять весь экран | `Shift + Cmd + 3` |
| Снять окно | `Shift + Cmd + 4`, затем `Пробел` |
| Снять часть экрана | `Shift + Cmd + 4` |
| Открыть панель Screenshot | `Shift + Cmd + 5` |

## Фрагмент инструкции

### Скриншот всего экрана

1. Нажмите `Shift + Cmd + 3`.
2. Найдите файл скриншота на рабочем столе.

Примечание: чтобы скопировать скриншот в буфер обмена, нажмите `Shift + Cmd + Control + 3`.

### Скриншот окна

1. Нажмите `Shift + Cmd + 4`.
2. Нажмите `Пробел`.
3. Наведите указатель камеры на нужное окно.
4. Нажмите кнопку мыши.

### Скриншот части экрана

1. Нажмите `Shift + Cmd + 4`.
2. Наведите курсор в начало области.
3. Зажмите кнопку мыши и выделите фрагмент.
4. Отпустите кнопку мыши.

## Что улучшает такая инструкция

- пользователь выбирает сценарий, а не ищет нужную кнопку в длинном описании;
- горячие клавиши вынесены в таблицу;
- каждый раздел содержит только один пользовательский результат;
- примечания отделены от обязательных шагов;
- структуру легко расширить для других платформ.
