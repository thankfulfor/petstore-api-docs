Changelog

Все значимые изменения в документации фиксируются на этой странице.

2026-04-11

Added

  • Добавлен раздел Changelog в структуру документации.
  • Добавлена страница истории изменений для портфолио-кейса.
  • Добавлена страница Версии API с политикой версионирования и шаблоном для v3.
  • Добавлен плагин поиска search в конфигурацию MkDocs.
  • Добавлен кастомный CSS docs/stylesheets/extra.css для управления отступами заголовков.

Changed

  • Обновлена структура API Reference: endpoints сгруппированы по ресурсам (Pet, Store).
  • В API Reference добавлен раздел про поиск и фильтрацию в документации.
  • В Getting Started исправлена разметка списков для корректного отображения по строкам.

Fixed

  • Исправлена локализация пагинации (Previous/Next) в теме Read the Docs.
  • Улучшен JS-хук локализации, чтобы перевод применялся после динамической перерисовки DOM.
  • Удалены лишние блоки с общими best practices с главной страницы.

2026-04-10

Changed

  • Проект переведен с плейсхолдерного TaskFlow API на публичный Swagger Petstore API.
  • Обновлены Home, Getting Started, API Reference и openapi/openapi.yaml под реальные публичные эндпоинты.

Added

  • Добавлены русские подписи пунктов меню в навигации документации.

2026-04-09

Added

  • Подготовлена базовая структура контента для API-документации: Home, Getting Started, API Reference.
  • Добавлены примеры запросов и ответы для быстрого старта.

Fixed

  • Исправлен CI-деплой GitHub Pages: добавлено разрешение contents: write в workflow.
