Версии API

Текущая версия

  • v2 (активная): Swagger Petstore API, описанная в текущей документации.

Поддерживаемые версии в документации

  • v2 - рабочая версия (текущая).
  • v3 - шаблон для следующей версии (пока в разработке).

Политика версионирования

  • Новая major-версия получает отдельный раздел документации.
  • Для каждой версии фиксируются поддерживаемые endpoints и отличия от предыдущей.
  • При релизе новой версии changelog должен содержать миграционные заметки.
